Overview

MyGame is a 2D platformer built with C and SDL. It supports solo and multiplayer modes, allowing players to control characters with default or red skins. Players can move, jump, sprint, attack, and use a shield power-up. The game features a menu for selecting game modes and input configurations.

Key Components

  • personnage: A structure representing a player character, including position, sprites, status, and power-up states.
  • main.c: Handles game initialization, menu, and game loop.
  • personne.c/h: Defines the personnage structure and functions for initialization, rendering, movement, and input handling.

Controls

  • Player 1: Arrow keys (move, jump), Shift (sprint), P (shield), G (guide), Left Click (attack).
  • Player 2: A/D (move), W (jump), Shift (sprint), P (shield), G (guide), Left Click (attack).
  • General: Esc (quit), G (toggle guide).

Dependencies

  • SDL, SDL_image, SDL_ttf, SDL_mixer
